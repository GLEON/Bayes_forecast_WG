4.2_Calibrate_Bayesian_models

Scripts to run Bayesian model calibration using data from 2009-2014

Guide to files:

4.2_Calibrate_Bayesian_models: Rscript that runs a for-loop to sequentially calibrate all Bayesian models found in the 4.1_JAGS_models folder from 2009-2014

Output is written to 5_Model_output --> 5.1_Calibration
