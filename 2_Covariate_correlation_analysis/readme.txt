2_Covariate_correlation_analysis

Scripts to run covariate correlation analysis to determine which covariates to include in Bayesian models

Guide to file structure:

2A_Covariate_correlation_analysis: Rscript to run correlations between G. echinulata density and hypothesized environmental drivers from 2009-2014

2B_Bayes_selected_variables_correlation: Rscript to run correlations between physical covariates for G. echinulata hindcast models to avoid inclusion of collinear covariates in a model
