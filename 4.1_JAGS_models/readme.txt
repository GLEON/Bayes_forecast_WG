4.1_JAGS_models

R files containing Bayesian models in JAGS format to run in rjags

Refer to Table 1 of the Lofton et al. 2020 manuscript for descriptions of model structure and covariates included in each model.

RW = random walk model with uninformed observation error prior; not presented in manuscript

RW_obs = random walk model with informed observation error prior; corresponds to RW model in manuscript

AR = AR

wtrtemp_min = MinWaterTemp

wtrtemp_min_lag = MinWaterTempLag

wtrtemp_MA7 = WaterTempMA

schmidt_med_diff = SchmidtMedDiff

schmidt_max_lag = SchmidtMaxLag

wind_dir_2day_lag = WindDir

precip = Precip

GDD = GDD

schmidt_and_precip = SchmidtAndPrecip

schmidt_and_wnd = SchmidtAndWind

wnd_and_precip = WindAndPrecip

wnd_and_GDD = WindAndGDD
