4.4_Uncertainty_partitioning

Scripts to run Bayesian model uncertainty partitioning for model validation from 2015-2016

Guide to files:

4.4_Uncertainty_partitioning: collates hindcast input for one-week-ahead through four-week-ahead forecasts and calculates relative contributions of each uncertainty source for all models in 2015-2016 hindcasts

Output is written to 5_Model_output --> 5.3_Uncertainty_partitioning
